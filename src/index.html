Reto CRUD para la Base de Datos del Hospital

Nivel 1: Hospital
1. Listar Hospitales: Muestra una lista de todos los hospitales en la base de datos. - ✅
2. Mostrar Hospital: Permite al usuario ingresar el ID de un hospital y muestra toda su información, incluidos los departamentos, médicos, pacientes, registros médicos y prescripciones. ✅
3. Crear Hospital: Permite al usuario ingresar el nombre de un nuevo hospital y lo agrega a la base de datos. ✅
4. Actualizar Hospital: Permite al usuario ingresar el ID de un hospital existente y actualizar su nombre. ✅
5. Eliminar Hospital: Permite al usuario ingresar el ID de un hospital y lo elimina de la base de datos. ✅
